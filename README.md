# 🎮 HTML Games

Simple browser-based games. No installs, no frameworks — just HTML, CSS, and vanilla JS.

## Games

### 🔢 Math Blitz
Add and subtract numbers up to 99 against the clock.
- 3 difficulty levels: Easy (60s, 0–9), Medium (45s, 0–50), Hard (30s, 0–99)
- Tracks correct answers, wrong answers, and accuracy
- Instant feedback per question

[Play →](math-blitz/index.html)

## Running Locally

Just open `index.html` in any browser — no server needed.

## Contributing

PRs welcome! Each game lives in its own folder with a self-contained `index.html`.
